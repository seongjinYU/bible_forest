# 팀 숲 성경읽기 웹앱 기획서

> 교회 대학부 | 2026 기획 제안  
> 대학부 성경 읽기 활성화를 위한 참여형 챌린지 플랫폼

---

## 1. 핵심 컨셉

**"함께 심고 함께 자라는 팀 숲"**

성경읽기표로 장을 체크해 인증하면, 우리 팀 숲에 나무가 자라는 **참여형 성경읽기 챌린지**

- 챌린지: 신약 1독 (6월 시작 ~ 수련회, 약 2달 반)
- 10장 읽을 때마다 랜덤 나무 1그루 획득 → 팀 숲에 직접 배치

| 가치 | 설명 |
|------|------|
| 기여와 독려 | 경쟁이 아닌 팀 전체 진행률에 기여하는 구조로 서로를 독려하며 함께 완주 |
| 여정의 연속성 | 6월부터 시작하여 여름 수련회까지 이어지는 체계적인 신앙 훈련의 장 마련 |

---

## 2. 기획 배경 — 왜 팀 숲인가

| 구분 | 현황 및 문제점 | 팀 숲의 해결책 |
|------|--------------|--------------|
| 참여 동기 | 개인 의지만으로는 지속 어려움, 중도 포기자 다수 발생 | 팀 전체 진행률에 기여하는 팀 미션 구조로 공동체적 책임감과 성취감 부여 |
| 연속성 | 일회성 이벤트로 끝나는 경우가 많아 신앙 훈련의 흐름이 끊어짐 | 온라인 챌린지 결과를 오프라인 수련회 프로그램과 연계하여 은혜의 연속성 확보 |

---

## 3. 대상 및 로그인 정책

### 타깃 사용자 (대학부)
- **디지털 네이티브**: 모바일에 익숙하고 직관적 인터페이스 선호
- **감성적 공유 선호**: 시각적 성취감과 정서적 교감에 동기부여
- **공동체 소속감**: 팀과 함께 목표를 이루는 과정에서 소속감과 재미를 느낌

### 로그인 정책 — Zero-Friction

진입 장벽을 최소화하여 참여율을 높이는 것이 핵심 원칙

- 이름 입력 + 팀 선택(1~7팀)만으로 즉시 등록
- 쿠키 저장 기술로 자동 로그인 유지
- 별도 회원가입/비밀번호 없는 간편 구조
- 동명이인 예외 처리 없는 직관적 운영

## 4. 서비스 흐름 — 씨앗부터 숲까지

```
1. 최초 접속       → 이름 입력 및 소속 팀 선택
2. 메인 화면       → 우리 팀 숲의 현재 상태 확인
3. 성경읽기표      → 신약 책/장 체크, 누적 장수 표시
4. 나무 획득       → 10장 달성 시 랜덤 나무 1그루 알림
5. 나무 배치       → 획득한 나무를 원하는 위치에 배치
6. 숲 업데이트     → 팀 전체 나무 배치 현황이 숲에 반영
7. 다른 팀 숲 보기 → 전체 팀 숲 비교
```

## 5. 주요 기능 상세

### 5-1. 메인 화면 — 팀 숲 대시보드

| 기능 | 설명 |
|------|------|
| 시각적 성장 시스템 | 팀원들이 배치한 나무가 숲 이미지에 실제로 표시되는 방식 |
| 실시간 현황 표시 | 현재 심어진 나무 수, 참여 인원, 팀 닉네임을 메인 상단에 직관적으로 노출 |
| 타 팀 둘러보기 | 다른 팀의 숲 상태를 확인하며 선의의 경쟁과 동기부여 유발 |

### 5-2. 성경읽기표 인증 시스템

| 기능 | 설명 |
|------|------|
| 장 단위 체크 | 신약 전체 책/장을 체크리스트로 표시, 개인별 누적 장수 표시 |
| 나무 획득 알림 | 10장 달성 시 랜덤 나무 1그루 획득 알림 |
| 신약 1독 보너스 | 신약 1독 완료 시 특별 나무 + 보너스 점수 지급 |

### 5-3. 나무 배치 화면

| 기능 | 설명 |
|------|------|
| 위치 지정 | 획득한 나무를 드래그 또는 탭으로 숲 내 원하는 위치에 배치 |
| 좌표 저장 | 배치 위치를 좌표로 DB에 저장하여 팀 숲에 반영 |

### 5-4. 어드민 대시보드

비전공자 관리자도 손쉽게 운영할 수 있는 직관적인 관리 환경 (어드민 전용)

| 기능 | 설명 |
|------|------|
| 어드민 로그인 | 관리자 전용 인증 |
| 챌린지 관리 | 챌린지 시작·종료일 설정, 기간별 목표 진행률 모니터링 및 상태 관리 |
| 팀 관리 | 소속 팀 목록 구성 및 팀별 고유 숲 테마 커스터마이징 |
| 전체 현황 대시보드 | 팀별 진도율, 개인별 인증 횟수 조회 |
