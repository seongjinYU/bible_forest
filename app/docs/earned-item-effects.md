# 아이템 획득 이펙트 스펙

> `EarnedItemDialog` — 등급(B/A/S) × 테마(숲/우주/바다) 파티클 이펙트 설계

---

## 이펙트 매트릭스

| 등급 | 확률 | 숲 (forest) | 우주 (night) | 바다 (ocean) |
|------|------|-------------|--------------|--------------|
| **B** | 65% | 효과 없음 (팝 애니메이션만) | 효과 없음 (팝 애니메이션만) | 효과 없음 (팝 애니메이션만) |
| **A** | 30% | 나뭇잎 살짝 터짐 | 별 반짝임 | 파도 |
| **S** | 5%  | 낙엽 폭우 → 줌인 → 폭발 | 별똥별 쏟아짐 → 줌인 → 폭발 | 소용돌이 → 줌인 → 폭발 |

---

## 등급별 상세

### B등급 — 효과 없음
카드 팝(pop) 애니메이션만 재생. 파티클 없음.

---

### A등급

#### 숲 A — 나뭇잎 살짝 터짐
- 아이콘 주변에서 초록 나뭇잎 18개가 짧게 퍼졌다 빠르게 사라짐
- 사거리 짧음 (반경 ~60px), 빠른 페이드아웃
- 색상: `#7BBF48` `#5A9E20` `#A8D55A` `#C0DD97`

#### 우주 A — 별 반짝임
- 화면 전체에 소형 별 38개가 랜덤 위치에서 깜빡이다 사라짐
- 빛 강도가 사인파로 진동 (`sin(phase)`)
- 큰 별(1.6px↑)에는 십자 빛줄기 추가
- 색상: `#AFA9EC` `#EEEDFE` `#B5D4F4` `#fff` `#ffd700`

#### 바다 A — 파도
- 좌우 양쪽에서 물방울 입자들이 물결치며 중앙을 향해 흘러옴
- 각 입자는 사인파 수직 진동으로 파도 질감 표현
- 색상: `#378ADD` `#85B7EB` `#5DCAA5` `#9FE1CB`

---

### S등급 — 공통 구조

> **줌인(~1.5초 천천히) → 콜백(폭발) → 줌아웃(~0.3초 빠르게)**  
> 폭발 시점에 아이콘 3D 스핀, 플래시, 링 애니메이션 동시 실행

#### 숲 S — 낙엽 폭우 → 폭발
- 줌인 동안: 나뭇잎 5개/20ms 간격으로 화면 상단 전체에서 빠르게 낙하 (vy=5~10px/frame)
- 줌아웃 트리거: 낙하 중이던 잎 전체가 반경 방향으로 폭발 튀어나감
- 추가 버스트: 90개 파티클 방사형 폭발
- 색상: `#97C459` `#639922` `#a8d55a` `#C0DD97` `#d4f0a0`

#### 우주 S — 별똥별 → 폭발
- 줌인 동안: 화면 위·좌·우 끝에서 중앙 부근을 향해 별똥별이 날아옴 (65ms 간격)
- 각 별똥별은 24프레임 잔상 트레일 보유, 흰색 헤드 + 컬러 트레일
- 줌아웃 트리거: 모든 별똥별이 이동 방향으로 퍼져나가며 폭발
- 추가 버스트: 85개 별/원형 파티클
- 색상: `#AFA9EC` `#B5D4F4` `#ffd700` `#fff`

#### 바다 S — 소용돌이 → 폭발
- 줌인 동안: 3개 링(반경 60·104·148px)에 84개 입자가 타원 궤도로 회전
- 깊이(dep = sin(angle))에 따라 뒤(cvB) / 앞(cvF) 캔버스 분리 → 3D 공전감
- 아이콘도 함께 Y축 회전 (3D 스핀)
- 줌아웃 트리거: 모든 공전 입자가 바깥쪽으로 폭발
- 색상: `#378ADD` `#5DCAA5` `#85B7EB` `#9FE1CB` `#1D9E75`

## 구현 메모

### 캔버스 레이어 구조

```
z-index: 2   <canvas id="cvB">   파티클 (다이얼로그 뒤)
z-index: 6   <div id="dim">      모달 딤 오버레이 (rgba 0→0.38)
z-index: 10  <div id="dm">       아이템 획득 다이얼로그 카드
z-index: 20  <canvas id="cvF">   파티클 (다이얼로그 앞)
```

S등급 소용돌이 파티클만 `dep = Math.sin(angle)` 값으로 cvB/cvF를 동적 배분.  
나머지 이펙트는 모두 cvF에만 렌더링.

### 아이콘 3D 스핀 (S등급 공통)
```
.ic-wrap { perspective: 140px }
ii.style.transform = `rotateY(${iconRotY}deg)`  // rAF 루프에서 매 프레임 갱신
```
폭발 시점에 초기 속도 15deg/frame → 감속하며 멈춤.

### 줌 타이밍
| 구간 | 증분/frame | 총 소요 |
|------|------------|---------|
| 줌인 | +0.011 | ~1.5초 |
| 줌아웃 | +0.058 | ~0.3초 |
최대 스케일: 1.24× (`scale(1 + t * 0.24)`)

### 정리 함수 `stopAll()`
컴포넌트 언마운트·닫기·다음 아이템 전환 시 반드시 호출:
- `cancelAnimationFrame` + `clearInterval(spawnTimer)`
- 파티클 배열 초기화, 캔버스 클리어
- 아이콘 transform 리셋, 딤 오버레이 투명화
